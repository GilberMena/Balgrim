# Balgrim Frontend Next

Base inicial para migrar desde el frontend actual a:

- Next.js
- React
- Zustand
- API backend Express

## Arranque

```bash
npm install
npm run dev
```

Configura `.env` a partir de `.env.example`.

## Objetivo

Migrar progresivamente:

1. catalogo
2. carrito
3. checkout
4. auth admin
5. account area
